## Projeto

Projeto web responsivo de uma acapsula do tempo para exibir memórias em uma timeline.

## Tecnologias

Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias.

- HTML
- CSS
- GIT e GITHUB
